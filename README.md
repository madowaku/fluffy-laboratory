# Fluffy Laboratory / 未定形研究室

**ふわふわしているが、研究である。**

Fluffy Laboratory は、論文の最後に残された課題、科学ニュースの余白、日常の違和感、数学パズルのタネを、  
「仮説の綿毛」として集めて育てるための小さな研究室です。

ここにあるものは、まだ確定した知識ではありません。  
でも、捨てるには惜しい問いです。

## 何をする場所か

- 論文末尾の Future Work を、人間向けのクエストにする
- 科学ニュースやメモから、未検証の仮説カードを作る
- 数学や論理の構造を、パズルのタネとして保存する
- 日常の違和感に名前をつける
- 「確定事実」「解釈」「仮説」「妄想」「危ない飛躍」を分ける

## MVP

最初のMVPは、URLやメモを入力して、AIがカード案を生成し、人間が編集・保存・MarkdownエクスポートできるWebアプリです。

## カード種別

- Hypothesis Card
- Future Work Quest
- Puzzle Seed
- Observation Card
- Note

## 合言葉

> 研究になる前の問いを、ふわふわのまま観察する。
